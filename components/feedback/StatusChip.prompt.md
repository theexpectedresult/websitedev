Pill status chip built on the ER Tech project-tracker palette — use it in tables, punch lists and dashboards.

```jsx
<StatusChip status="complete">Installed</StatusChip>
<StatusChip status="progress">In progress</StatusChip>
<StatusChip status="parts">Needs parts</StatusChip>
<StatusChip status="blocked">Blocked</StatusChip>
```

Statuses: `complete` (green) · `progress` (cream) · `parts` (peach) · `blocked` (red) · `info` (cyan) · `neutral` (slate). Set `dot={false}` to drop the leading dot.
